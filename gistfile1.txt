### Sudoku
Your task is to write a Sudoku class, so all tests will pass

### What the Sudoku is about?
Sudku is 9x9 puzzle pre-filled with some digits. 
The goal is to fill all the empty spaces, and meet all below requirements:
- the digits are unique in each row
- the digits are unique in each column
- the digits are unique in each of the 3x3 block 

### How to run?
Install rspec
```
gem install rspec
```

Run sudoku specs
```
rspec sudoku_spec.rb
```
